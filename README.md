**Red Consistency** is a multidisciplinary luxury real estate platform designed for the Egyptian market. It bridges the gap between property investment and architectural sanctuary through a **Complete Cycle** of services.

---

## 🏛️ The Three Pillars

- **The Collections (New Compounds):** Curated access to elite off-plan developments in New Cairo and Zayed.  
- **The Exchange (Marketplace):** A private C2C resale circuit for verified luxury residences.  
- **The Atelier (Renovation):** Professional finishing and interior artistry, from marble flooring to custom lighting.  

---

## ✨ Key Features

- **Mosaic Collections Grid** – A non-uniform, narrative portfolio layout for high-end compounds.  
- **Luxury Marketplace** – A 3-column responsive grid with icon-based property specs (beds, baths, area).  
- **Glassmorphism UI** – A sophisticated **Teal & Dark Blue** palette with blurred glass effects.  
- **Dynamic Routing** – Integrated detail pages for every listing, compound, and service line.  
- **Responsive Z-Pattern Storytelling** – A cinematic About page documenting the brand story and philosophy.  

---

## 🎨 Design System

- **Colors**
  - Deep Teal: `#052A35`
  - Signature Teal: `#09637E`
  - Accents: `#7AB2B2` and soft off-whites for typography and glass overlays.

- **Typography**
  - **Headings:** `CaskoLuxury` (luxury display, uppercase, increased letter-spacing)  
  - **Body:** `Inter` (or equivalent modern sans-serif)

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Routing:** React Router  
- **Animations:** Framer Motion (staggered reveals, smooth page transitions)  
- **Styling:** CSS3 (Flexbox/Grid, glassmorphism, custom layouts)  
- **Tooling / Runtime:** Node.js, npm

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/SeifAsran/Red_Consistency_real_estate.git
cd Red_Consistency_real_estate
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm start
```

The app will be available at `http://localhost:3000` (by default).

### 4. Build for production

```bash
npm run build
```
